## 🦉 우리는 누구인가요?

> **아울게임즈단**은 성공회대학교 IT동아리 **SOUL** 안의 게임 개발 스터디 그룹입니다.

유니티(Unity) 엔진을 기반으로 게임 제작을 함께 배우고,  
서로의 프로젝트를 공유하며 성장하는 **게임 개발 커뮤니티**예요. 🎮

## 🗺️ 뉴비 맞춤 커리큘럼

> 처음 시작하는 분도 따라올 수 있도록 **단계별로** 진행해요!

```mermaid
flowchart LR
    A([🌍 STEP 1<br/>월드 구축<br/>캐릭터 컨트롤]) --> B([❤️ STEP 2<br/>캐릭터 상태<br/>정보 시스템])
    B --> C([🌐 STEP 3<br/>멀티플레이<br/>행동 동기화])
    C --> D([🎯 STEP 4<br/>아이템 &<br/>레이캐스트])
    D --> E([📦 STEP 5<br/>오브젝트<br/>상호작용])
    E --> F([🤖 STEP 6<br/>AI 시스템<br/>NPC & Mobs])

    style A fill:#7C3AED,stroke:#fff,color:#fff
    style B fill:#8B5CF6,stroke:#fff,color:#fff
    style C fill:#A78BFA,stroke:#fff,color:#fff
    style D fill:#C4B5FD,stroke:#fff,color:#000

    style F fill:#EDE9FE,stroke:#fff,color:#000
```

<details>
<summary><b>📖 커리큘럼 상세 보기 (클릭)</b></summary>

| 단계 | 주제 | 학습 내용 |
|:---:|:---|:---|
| **1️⃣** | 월드 구축 & 캐릭터 컨트롤 | 씬 구성, Player Controller, Camera Follow |
| **2️⃣** | 캐릭터 상태 정보 | HP/MP, 스탯 시스템, ScriptableObject 활용 |
| **3️⃣** | 멀티 접속 & 동기화 | 네트워크 기초, 행동 동기화 구현 |
| **4️⃣** | 아이템 & 레이캐스트 | 인벤토리, Raycast 기반 인터랙션 |
| **5️⃣** | 오브젝트 상호작용 | 트리거, 이벤트 시스템 |
| **6️⃣** | AI 만들기 | NPC 대화, 몬스터 AI, Behaviour Tree 입문 |

</details>
