### 🤝 How to Join & Contribute
1. **Repository Exploration:** 관심 있는 스터디나 프로젝트 저장소를 살펴보세요.
2. **Fork & PR:** 본인만의 폴더를 만들어 학습 내용을 정리하고 `Pull Request`를 보내주세요.
3. **Code Review:** 동료들의 PR에 의견을 남기며 서로의 성장을 돕습니다.
4. **New Idea:** 새로운 스터디 주제가 있다면 언제든 [Discussions] 탭에 제안해 주세요!
